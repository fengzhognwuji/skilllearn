# Java基础语法演示

## 核心知识点

### 1. 数据类型

- 基本类型：int, double, boolean, char等
- 引用类型：String, 数组, 对象
- 类型转换：自动转换和强制转换

### 2. 控制语句

- 条件语句：if-else, switch-case
- 循环语句：for, while, do-while
- 跳转语句：break, continue, return

### 3. 面向对象基础

- 类与对象
- 封装、继承、多态
- 构造方法
- this和super关键字

### 4. 异常处理

- try-catch-finally
- throws关键字
- 自定义异常

### 5. 集合框架

- List/Set/Map接口
- ArrayList/HashSet/HashMap实现类
- 泛型使用

## 项目演示内容

```java
// 基本变量
int num = 10;
String text = "Hello";

// 条件语句
if(num > 5) {
    System.out.println("大于5");
}

// 方法定义
public static int add(int a, int b) {
    return a + b;
}
```
